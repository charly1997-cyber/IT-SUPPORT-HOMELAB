# Project 01 - Active Directory Lab

## Objective
Create and manage Active Directory users and Organizational Units (OUs).

## Lab Environment
- Windows Server
- Active Directory
- Windows Client
- VMware

## Tasks Performed
- Installed Active Directory Domain Services
- Promoted server to Domain Controller
- Created Organizational Units (OUs)
- Created user accounts
- Managed user passwords
- Tested account login

## Skills Demonstrated
- Active Directory Administration
- User Management
- Account Management
- Windows Server Administration

## Outcome
Successfully created and managed Active Directory users and OUs in a Windows Server lab environment
